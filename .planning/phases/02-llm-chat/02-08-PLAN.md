<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete LLM Chat module:
- Two-column layout: conversation list (left, resizable) + chat view (right)
- Real-time streaming via Tauri Channel API
- Conversation history persisted in SQLite
- Model switching per message (full history sent to new model)
- File attachments (PDF/images) sent as base64 inline
- Markdown rendering with syntax highlighting and copy button
- Conversation management: create, search, time-grouped list, delete with confirmation
- Keyboard shortcuts: Enter sends, Shift+Enter newlines, Cmd/Ctrl+K new conversation
  </what-built>
  <how-to-verify>
Start the app in dev mode: `npm run tauri dev`

**Test 1 — Basic streaming (CHAT-01):**
1. Open app → click Chat icon in sidebar (should be enabled now, not greyed out)
2. Click "Nova Conversa"
3. Type "Olá! Descreva o que é recursão em programação." and press Enter
4. Expected: tokens stream token-by-token in real-time; NOT a bulk response dump
5. Expected: "Parar" button appears during streaming
6. Expected: After stream ends, Regenerate button appears on AI message

**Test 2 — Conversation history (CHAT-02):**
1. Close and reopen the app (or press Cmd+R to reload)
2. Expected: Previous conversation appears in conversation list under "Hoje"
3. Click on the conversation
4. Expected: Full message history appears (both user message and AI response)
5. Type a follow-up message: "Me dê um exemplo em Python."
6. Expected: AI responds with context of the previous message (has recursion context)

**Test 3 — Model switching (CHAT-03):**
1. In the same conversation, change the model in the dropdown
2. Send another message: "Agora explique de forma mais simples."
3. Expected: Third message is answered by the new model
4. Expected: Each AI message shows the model badge (which model answered)
5. Expected: The new model's response has full context (sees all previous messages)

**Test 4 — File attachment (CHAT-04):**
1. Click the paperclip icon
2. Select a PNG image or PDF file from your system
3. Expected: File preview appears above input (image thumbnail or doc icon + filename)
4. Type "O que está nesta imagem?" or "Descreva este documento."
5. Press Enter to send
6. Expected: AI response references the file content

**Test 5 — Markdown + code blocks (CHAT-05):**
1. Send: "Escreva uma função em Python para calcular fibonacci com recursão."
2. Expected: Code block appears with `python` language badge
3. Hover over the code block
4. Expected: "Copiar" button appears (not visible without hover)
5. Click "Copiar"
6. Expected: Button changes to "Copiado!" for 2 seconds

**Test 6 — Search and delete (CHAT-02 + D-29):**
1. Create 3-4 conversations with different topics
2. Type a keyword from one conversation in the search box
3. Expected: List filters in real-time to matching conversations
4. Right-click a conversation
5. Expected: Context menu shows "Excluir" option
6. Click Excluir → confirm in dialog
7. Expected: Conversation disappears from list

**Test 7 — Keyboard shortcuts:**
1. Press Cmd+K (macOS) or Ctrl+K (Windows)
2. Expected: New conversation created and focused
3. Type a message with multiple lines using Shift+Enter
4. Expected: Input grows, Enter does NOT send mid-paragraph
  </how-to-verify>

Type "aprovado" if all tests pass.

  </resume-signal>
